MDPM

MDPM is an open source framework for Project Management based on Eclipse. It consists of a set of domain specific languages enabling the modeling of Work Breakdown Structures (WBS), Scheduling and Gantt diagrams, Resources allocation, etc.

DoNo

DoNo (Document Notation) is a set of Eclipse plugins realizing editor and parser for a DSL aimed at representing the main components of a document. Automatic generation workflows are also available enabling the automatic generation of Word docs out of DoNo models.
